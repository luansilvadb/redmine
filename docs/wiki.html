Wiki Operacional do Sistema

Objetivo

Este documento descreve regras, fluxos e responsabilidades dos agentes do sistema. Seu conteúdo é derivado do arquivo wiki.xml.

Agente: Integrações

Responsabilidades

  • Validar contratos de API REST
  • Garantir versionamento correto
  • Monitorar falhas de integração

Limitações

  • Não alterar dados em produção
  • Não responder fora do escopo de integrações

Fluxo de Validação

  1. Receber requisição
  2. Validar schema
  3. Executar testes
  4. Responder resultado

Erros Comuns

  • Token expirado
  • Endpoint incorreto
